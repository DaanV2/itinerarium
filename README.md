# Itinerarium

A self-hosted web tool for tabletop RPG campaigns. Supports both players and game masters with character management, journaling, group organisation, and a game-day-aware documentation system.

## Documentation

- [Onboarding](docs/onboarding.md) — start here: setup, running the stack, the mental model
- [Features](docs/features.md) — players, GMs, groups, sessions, locations
- [Architecture](docs/architecture.md) — data model, permission rules, tech stack
- [Currency & Money](docs/currency.md) — currencies, conversion ratios, and character money
- [Configuring Currencies](docs/currency-configuration.md) — the catalog file and runtime API
- [Deployment](docs/deployment.md) — Docker Compose setup and configuration
- [Roadmap](docs/roadmap.md) — alpha/beta milestones
- [Backlog](docs/backlog.md) — open ideas not yet designed

```bash
docker compose up -d
```
